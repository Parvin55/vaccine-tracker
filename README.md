# Vaccine Tracker (Immunization Log)

A responsive web application that allows users to manage a digital immunization record.  
The system tracks vaccines taken by the user and intelligently calculates the **next dose or booster date**, displaying the current status as **Upcoming**, **Due Today**, or **Overdue**.

The Vaccine Tracker helps users maintain a structured and reliable immunization history.  
After logging in with predefined credentials, users can:

- Select a vaccine
- Enter the dose date
- Automatically calculate the next dose date
- View vaccine status
- Edit or delete records
- Log out securely

The application is fully responsive and works on both **desktop and mobile devices**.
